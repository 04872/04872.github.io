整除关系

带余除法 对于 $P[x]$ 上的任意两个多项式 $f(x)$ 和 $g(x)$，其中 $g(x)\neq 0$，则一定有 $P[x]$ 中的多项式 $q(x)$、$r(x)$ 存在，使得 $$f(x) = q(x)g(x)+r(x).$$ 证明 : 过程就是带余除法的过程.

定义1 称数域 $P$ 上的多项式 $g(x)$ 整除 $f(x)$ ，如果有数域 $P$ 上的多项式 $h(x)$ 使得 $$f(x) = g(x)h(x).$$ 我们用 “$g(x)\mid f(x)$” 表示 $g(x)$ 整除 $f(x)$，用 “$g(x)\nmid f(x)$” 表示 $g(x)$ 不整除 $f(x)$.

整除关系不随数域的改变而改变

定理1 对于数域 $P$ 上的任意两个多项式 $f(x)$，$g(x)$，其中 $g(x)\neq 0$, $g(x)\mid f(x)$ 的充要条件是 $g(x)$ 除 $f(x)$ 的余数是零.

证明 充分性：如果 $r(x)=0$，那么 $f(x) = q(x)g(x)$，所以整除. 必要性：根据整除定义，$f(x) = q(x)g(x)=q(x)g(x)+0$.

整除性质1 如果 $f(x)$ 和 $g(x)$ 相互整除，那么存在非零常数 $c$，使得 $f(x)=cg(x)$.

证明 有 $f(x)h_1(x) = g(x)$，$g(x)h_2(x)=f(x)$.

​ $\Rightarrow f(x)h_1(x)h_2(x) = f(x)$. ​ 如果 $f(x)=0$，那么 $g(x) = 0$，命题显然成立.

​ 如果 $f(x)\neq 0$，由多项式乘法消去律消去 $f(x)$ 得，$h_1(x)h_2(x) = 1$，那么 $\partial(h_1(x))+\partial(h_2(x)) = 0$.

所以 $\partial(h_1(x)) = \partial(h_2(x))=0$. 那么 $c$ 取 $h_2(x)$ 即可.

整除性质2（传递性） 如果 $f(x)\mid g(x)$，$g(x)\mid h(x)$，那么 $f(x)\mid h(x)$.

证明 根据整除定义简单验证即可.

整除性质3 如果 $f(x)\mid g_i(x),\quad i=1,2,3,\cdots,r$，那么

$$f(x)\mid \sum_{i=1}^{r}u_i(x)g_i(x).$$

其中 $u_i(x)\in P[x]$.

证明 对任意 正整数$1\leq i\leq r$，存在 $h_i(x)\in P[x]$，使得 $g_i(x) = f(x)h_i(x)$.

所以 $\sum_{i=1}^{r}u_i(x)g_i(x) = \sum_{i=1}^{r}u_i(x)h_i(x)f(x) = f(x)\sum_{i=1}^{r}u_i(x)h_i(x)$.

最大公因式

最大公因式定义：

设 $f(x),g(x),u(x)\in f[x]$，如果 $u(x)$ 满足以下条件，那么称其为 $f(x)$ 与 $g(x)$ 的最大公因式，

  • $u(x)\mid f(x), u(x)\mid g(x)$.

  • 如果 $v(x)\mid f(x)$ 且 $v(x)\mid g(x)$，那么 $v(x)\mid u(x)$.

如何求两多项式之间的最大公因式呢？我们可以根据以下定理来求解

定理1

如果 $f(x)=q(x)g(x)+r(x)$，那么 $f(x)$ 与 $g(x)$ 的最大公因式和 $g(x)$ 与 $r(x)$ 的相同.

证明：设 $d(x)$ 是 $f(x)$ 和 $g(x)$ 的最大公因式，那么 $d(x)\mid r(x)$，所以 $d(x)$ 是 $g(x)$ 和 $r(x)$ 之间的公因式.

设有 $d_1(x)\mid r(x),d_1(x)\mid g(x)$，那么 $d_1(x)\mid f(x)$，所以 $d_1(x)$ 是 $f(x)$ 与 $g(x)$ 的因式，所以 $d_1(x)\mid d(x)$.

定理2

如果 $f(x),g(x)\in F[x]$，$d(x)$ 是二者的最大公因式，那么存在 $u(x),v(x)\in F[x]$，使得

$$f(x)u(x)+g(x)v(x)=d(x).$$

证明：证明过程就是利用定理1求解最大公因式的过程.

为了描述方便，我们一般记 $(f(x),g(x))$ 为 首项系数为 $1$ 的最大公因式.

互素

$f(x),g(x)\in F[x]$ 互素 $\triangleq (f(x),g(x))=1$.

互素的多项式之间有一些特殊的性质。

定理3

$\left(f(x),g(x)\right)=1 \Leftrightarrow \exists u(x), v(x)$，使得 $u(x)f(x)+v(x)g(x)=1$.

证明：首先证明必要性。根据 定理2，显然成立。再证明充分性，设 $d(x)=(f(x),g(x))$，则 $d(x)\mid 1$，而且显然 $1$ 是公因式，那么 $(f(x),g(x))=1$.

定理4

如果 $(f(x),g(x))=1$ 且 $f(x)\mid g(x)h(x)$，那么 $f(x)\mid h(x)$.

证明：$(f(x),g(x))=1$，那么 $\exists u(x),v(x)$，使得 $u(x)f(x)+v(x)g(x)=1$.

那么，$u(x)f(x)h(x)+v(x)g(x)h(x)=h(x)$.

所以 $f(x)\mid h(x).$
